Class Subjects

Cybersecuirty

The protection of internet-connected systems such as hardware, software and data from cyber-threats. The practices used by individuals and enterprises to protect against unauthorized access to data centers and other computerized systems.

Programming

Programming is the process of creating a set of instructions that tell a computer how to perform a task.Programming can be done using a variety of computer programming languages, such as JavaScript, Python, C, C#, C++, and Java.

System Administration

The field of work in which someone manages one or more systems, be they software, hardware, servers or workstations to ensure the systems are running efficiently and effectively.

Devops

A set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the systems development life cycle and provide continuous delivery with high software quality.

Database Administration

Ensuring that a database is always available as needed. Other closely related tasks and roles are database security, database monitoring and troubleshooting, and planning for future growth.

Data Science

Concept to unify statistics, data analysis and their related methods in order to understand and analyze actual phenomena with data drawn from mathematics, statistics and computer science.
